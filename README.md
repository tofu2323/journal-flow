# Journal Flow 🌊

マインドフルネス ジャーナリング PWA

## 機能

- 🧘‍♂️ フォーマル実践の記録
- 🚶 インフォーマル実践の記録  
- ☀️ 快な出来事の記録
- ☁️ 不快な出来事の記録
- 🗣️ 困難なコミュニケーションの振り返り

## 開発

```bash
npm install
npm run dev
```
